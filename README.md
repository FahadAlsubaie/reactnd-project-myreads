This is a bookshelf app which allows you to select books you want to read, read , or currently reading. This project is built with react and works with an API server which is provided by udacity.

## Quick Start

### To run locally:

```
git clone https://github.com/FahadAlsubaie/reactnd-project-myreads
cd reactnd-project-myreads
npm install
npm start
```

### Functionality

In this app , the root page shows a three categories, each one may or may not contains list of books. The three categories are:

- Currently Reading
- Want to Read
- Read

also you can search by clicking on the add button to add books.
